# ✈️ Trabook - Travel Agency Landing Page

A simple, responsive Travel Agency Landing Page built from a Figma design using pure HTML and CSS.

## 🎨 Features

- **Hero Section:** Intro header with search widget.
- **Things to do:** Services offered by the agency.
- **Deals & Discounts:** Travel packages and offers.
- **Vacation Plans:** Trip packages with details.
- **Testimonials:** Customer feedback layout.
- **Latest Blog:** Grid of travel blog posts.
- **Newsletter:** Subscription form layout.

## 🛠️ Tech Stack

- **HTML5**
- **CSS3** (Flexbox & Grid)

## 📁 Project Structure

```text
├── index.html          # HTML structure of the landing page
├── styles.css          # Styling and responsive layouts
├── Image/              # Photo assets
└── Vector-image/       # SVG logos and icons
```

## 🚀 How to Run

1. Clone or download this repository.
2. Open `index.html` in any web browser.
